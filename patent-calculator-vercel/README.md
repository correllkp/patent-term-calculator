# Patent Term Calculator - Vercel Version

A modern, interactive patent term calculator for calculating expiration dates and maintenance fee deadlines.

## Features

- Calculate patent expiration dates for utility, design, and plant patents
- Handle domestic benefit claims (continuations, CIPs, divisionals)
- Calculate maintenance fee deadlines
- Support for terminal disclaimers
- Responsive, modern UI with animations
- SEO optimized with schema markup

## Deployment to Vercel

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Deploy:
```bash
cd patent-calculator-vercel
vercel
```

3. Follow the prompts to link to your Vercel account

## Development

```bash
npm install
npm run dev
```
